# 高校教学管理软件

一个基于 Java Servlet + JSP + MySQL 的高校教学管理系统，实现了学生、教师、课程、成绩等核心教学管理功能。

## 项目简介

本系统采用经典的 MVC 设计模式，为高校教务管理提供了一套完整的解决方案。系统支持多角色权限管理，包括管理员、教师和学生三种角色，实现了从学生信息管理到成绩统计分析的完整教学管理流程。

## 技术栈

- **后端框架**：Java Servlet
- **前端技术**：JSP、HTML、CSS、JavaScript
- **数据库**：MySQL 8.0+
- **开发工具**：Eclipse
- **服务器**：Apache Tomcat 10.0.27
- **其他依赖**：
  - JSTL 2.0.0
  - jQuery 1.8.3
  - MySQL Connector/J 8.4.0
  - Commons BeanUtils、Collections、Lang
  - JSON-lib 2.4
  - JSmartUpload
  - JXL（Excel 导出）

## 核心功能模块

### 1. 用户权限管理
- **角色管理**：支持管理员、教师、学生三种角色
- **权限控制**：基于角色的菜单权限管理
- **登录认证**：用户登录验证和会话管理

### 2. 学生管理
- 学生信息增删改查
- 学生信息分页查询
- 学生信息统计
- 班级同学查询

### 3. 教师管理
- 教师信息增删改查
- 教师信息分页查询
- 教师授课班级查询

### 4. 班级管理
- 班级信息增删改查
- 班级信息分页查询

### 5. 专业管理
- 专业信息增删改查
- 专业信息分页查询

### 6. 课程管理
- 课程信息增删改查
- 课程信息分页查询

### 7. 班级课程管理
- 班级课程关联管理
- 课程安排查询
- 课程数据导出（Excel）

### 8. 成绩管理
- 成绩录入和修改
- 成绩查询和统计
- 成绩报表生成

### 9. 成绩统计分析（新增功能）
- **课程平均分统计**：统计各门课程的平均成绩和学生人数
- **分数段人数统计**：按分数段统计人数分布（不及格、60-69、70-79、80-89、90-100）
- **班级成绩分析**：按班级查看各课程的平均分、最高分、最低分和参考人数

## 数据库配置

### 数据库连接配置

数据库连接信息在 `cn.nwnu.aics.util.DB` 类中配置：

```java
private static final String URL = "jdbc:mysql://localhost:3306/your_database";
private static final String USER = "your_username";
private static final String PASSWORD = "your_password";
```

### 主要数据表

- `operator` - 操作员表（用户信息）
- `role` - 角色表
- `privilege` - 权限表
- `student` - 学生表
- `teacher` - 教师表
- `classes` - 班级表
- `major` - 专业表
- `subject` - 课程表
- `cla2sub` - 班级课程关联表
- `score` - 成绩表

## 部署说明

### 环境要求

- JDK 8 或更高版本
- Apache Tomcat 10.0.27
- MySQL 8.0+
- Eclipse IDE（推荐）

### 部署步骤

1. **导入项目**
   - 将项目导入到 Eclipse IDE
   - 确保项目配置为 Dynamic Web Project

2. **配置数据库**
   - 创建 MySQL 数据库
   - 执行数据库初始化脚本
   - 修改 `DB.java` 中的数据库连接信息

3. **配置 Tomcat**
   - 在 Eclipse 中配置 Tomcat 服务器
   - 将项目部署到 Tomcat

4. **启动服务**
   - 启动 Tomcat 服务器
   - 访问 `http://localhost:8080/Student/login.jsp`

## 使用说明

### 默认登录账户

根据数据库中的初始数据，通常有以下默认账户：

- **管理员**：用户名 `admin`，密码 `admin`
- **教师**：用户名 `teacher`，密码 `teacher`
- **学生**：用户名 `student`，密码 `student`

### 主要操作流程

1. **登录系统**
   - 访问登录页面
   - 输入用户名和密码
   - 系统根据角色显示相应菜单

2. **学生管理**
   - 管理员可以添加、修改、删除学生信息
   - 支持按学号、姓名、班级等条件查询

3. **成绩管理**
   - 教师可以录入和修改学生成绩
   - 支持按学生、课程、班级等条件查询成绩

4. **成绩统计分析**
   - 管理员和教师可以访问成绩统计功能
   - 查看课程平均分、分数段分布、班级成绩分析

## 技术特点

### 1. MVC 设计模式
- **Model**：Entity 和 DAO 层负责数据访问
- **View**：JSP 页面负责数据展示
- **Controller**：Servlet 负责业务逻辑控制

### 2. 过滤器机制
- **编码过滤器**：统一处理请求和响应的字符编码
- **登录过滤器**：保护受保护的页面，确保用户已登录

### 3. 分页查询
- 所有查询功能都支持分页显示
- 使用 AJAX 实现无刷新分页

### 4. 数据导出
- 支持班级课程数据导出为 Excel 文件
- 使用 JXL 库实现 Excel 文件生成

### 5. 数据可视化
- 成绩统计功能使用条形图展示数据
- 直观显示成绩分布情况

## 开发规范

### 代码结构
- 所有 Servlet 都继承 `HttpServlet`
- DAO 模式实现数据访问
- 统一的异常处理机制

### 命名规范
- 类名使用大驼峰命名法
- 方法名和变量名使用小驼峰命名法
- 常量使用全大写下划线分隔

### 注释规范
- 所有公共方法都添加 JavaDoc 注释
- 关键业务逻辑添加行内注释

## 常见问题

### 1. 数据库连接失败
- 检查 MySQL 服务是否启动
- 确认数据库连接信息是否正确
- 检查 MySQL 驱动是否正确配置

### 2. 页面显示乱码
- 确认数据库字符集为 UTF-8
- 检查编码过滤器是否正常工作

### 3. 权限访问问题
- 确认用户角色是否正确
- 检查权限表数据是否完整
